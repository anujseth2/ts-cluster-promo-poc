ThoughtSpot · System Architecture

Streamlit is the hub — everything moves through it

The two ThoughtSpot clusters and GitHub never talk to each other. The Streamlit app is the only thing connected to all of them, so content round-trips through it by two different paths: TML + feedback travel via Git, while NL instructions are read from source and written to target directly by API — never touching Git.
